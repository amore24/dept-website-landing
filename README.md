# Dept Website Landing Page
# This project is implemented by using react, redux toolkit, tailwind css and webpack

To Run this project please install node_module first by running `npm install` command.

After running the command to run json server run below command - 
`npx json-server --watch db.json --port 8000`
Make sure it is running on this url - http://localhost:8000/listingData

After that please run React application by below command - 
`npm start`

The web application will be running on this url - http://localhost:3000/
